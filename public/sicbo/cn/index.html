骰宝游戏规则
获胜条件
玩家猜测骰子开出的点数或不同骰子组合，把筹码放在游戏台上的不同投注区。如押注的内容与三个骰子停留开出的点数或组合相同​​，便赢得彩金。

操作及下注指南
  1. 开局后，下注倒数开始
  2. 每个单注金额不能高于或低于已设置个人限红之上、下限
  3. 若连续5局未下注，则会被强制退出游戏台
  4. 倒数结束后，游戏台会停止下注，而已确认之注码均不得更改或取回
  5. 停止下注后，由荷官启动机器自动摇骰
  6. 当摇骰停止，取三个骰子停留开出的点数及组合计算，是否与玩家押注的内容相同，来判定输赢
  7. 如摇骰停止后，任何一个骰子没有完全接触骰盅底面，或出现重叠状态，将会再次触发摇骰，以重新获取结果。

指定骰子
根据所选择点数出现的个数判断：

有一个骰子符合，则为「单骰」

有二个骰子符合，则为「双骰」

有三个骰子符合，则为「三骰」
单骰：1 赔 1
双骰：1 赔 2
三骰：1 赔 3
对子 ( 双骰、长牌 )
三个骰子中，任意两个骰子点数相同即为「对子」 ( 如：双 1 点 )
